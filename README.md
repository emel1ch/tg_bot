# 🏥 Цифровой помощник «Иду к врачу» 

[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://t.me/your_bot_link)
[![React](https://img.shields.io/badge/Frontend-React-61DAFB.svg)](https://reactjs.org/)
[![FastAPI](https://img.shields.io/badge/Backend-FastAPI-009688.svg)](https://fastapi.tiangolo.com/)

Комплексный сервис (Telegram-бот + Web Mini App), разработанный для детей с расстройствами аутистического спектра (РАС) и их родителей. 

**Главная цель проекта** — снизить тревожность ребенка перед медицинскими процедурами (поход к стоматологу, сдача крови) за счет предварительной визуализации, геймификации и социальных историй, а также обеспечить удобный интерфейс записи на прием к врачу.

---

## 🌟 Основной функционал

### 🤖 Telegram-бот (Пользовательский интерфейс)
* **Онбординг:** Приветствие и выдача согласия на обработку ПД.
* **Умная запись:** Пошаговая воронка записи (Город -> Клиника -> Врач -> Дата -> Время) с защитой от двойного бронирования (soft-hold).
* **Личный кабинет:** Просмотр статусов записей (ожидает, оплачено), отмена и перенос приема.
* **Поддержка:** Форма обратной связи с прямой пересылкой сообщений администратору.

### 📱 Telegram Mini App (Контент и Оплата)
* **Адаптационные материалы:** Мультфильмы, интерактивные игры-тренажеры, социальные истории (пошаговые визуальные инструкции) и PDF-рекомендации для родителей.
* **Оплата (Paywall):** Интеграция с ЮKassa для покупки платных курсов подготовки с автоматическим обновлением доступов через Webhooks.
* **In-place Admin Panel:** Встроенный поверх пользовательского UI режим администратора для редактирования контента, загрузки медиа и управления курсами.

### 🛠 Панель администратора (Бот)
* Автоматическая авторизация по Telegram ID.
* CRUD-операции со справочниками базы данных (добавление/удаление городов, клиник, врачей).
* Получение push-уведомлений о новых записях.

## 🏗 Стек технологий

**Backend:**
* Python 3.10+
* [aiogram 3.x](https://docs.aiogram.dev/) (Telegram Bot API)
* [FastAPI](https://fastapi.tiangolo.com/) (API для Mini App и обработка Webhooks)
* PostgreSQL + SQLAlchemy (База данных)
* ЮKassa API (Платежный шлюз)

**Frontend (Mini App):**
* React 18
* Vite
* TypeScript
* Telegram Web App SDK (`@twa-dev/sdk`)
